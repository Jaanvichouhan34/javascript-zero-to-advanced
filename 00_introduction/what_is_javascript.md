# What is JavaScript?

JavaScript is a high-level, interpreted programming language used to make web pages interactive and dynamic.

Initially, JavaScript was created to run only in web browsers. Today, it is used for:
- Frontend development (React, Angular, Vue)
- Backend development (Node.js)
- Mobile apps
- Desktop applications
- Game development

---

## Why JavaScript is Important?

- It is the **core language of the web**
- Supported by all modern browsers
- Used in both **frontend and backend**
- Huge job demand in Web Development and MERN Stack

---

## JavaScript is NOT Java

Despite the name, JavaScript and Java are completely different languages.

| JavaScript | Java |
|----------|------|
| Interpreted | Compiled |
| Dynamically typed | Statically typed |
| Used mainly for web | Used for enterprise apps |

---

## Where JavaScript Runs?

- Browser (Chrome, Firefox, Edge)
- Server (Node.js)
